# Install Weaviate with Docker Compose

Docker Compose uses a file named docker-compose.yml to define services.

This file must live in the root of the repository (next to README.md).

## Start Weaviate

```bash
docker compose up -d
```

## Verify container is running

```bash
docker ps
```
